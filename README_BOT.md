# Нейросетевой Telegram бот "Виталя"

Это Telegram бот, который использует нейронную сеть для генерации ответов в групповых чатах. Бот реагирует только тогда, когда его упоминают с помощью @ или по имени (Виталя, Витек, Виталий и т.д.), и отвечает в грубой и веселой форме, в зависимости от контекста.

## Особенности

- Использует нейросеть (RuGPT) для генерации ответов
- Отвечает только при упоминании (@username) или по имени
- Может отвечать в грубой или веселой манере
- Учитывает контекст сообщения при формировании ответа
- Работает в групповых чатах

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Получите токен для бота:
   - Напишите @BotFather в Telegram
   - Создайте нового бота командой /newbot
   - Следуйте инструкциям для получения токена

3. Создайте файл `.env` в корне проекта и добавьте токен бота:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

4. Запустите бота:

```bash
python run_bot.py
```

## Настройка

Конфигурация бота находится в файле `config/bot_config.py`, где можно изменить:

- `RUDE_RESPONSE_PROBABILITY` - вероятность использования грубых фраз
- `FUNNY_RESPONSE_PROBABILITY` - вероятность использования веселых фраз
- `GENERATION_TEMPERATURE` - температура генерации (как креативным будет ответ)
- Списки грубых и веселых фраз
- Ключевые слова, влияющие на стиль ответа

## Использование

1. Добавьте бота в групповой чат
2. Упомяните его в сообщении с помощью @ (например, "@bot_name Как дела?") или по имени (например, "Виталий, как дела?")
3. Бот сгенерирует ответ в грубой и/или веселой форме

## Примечания

- Для работы бота требуется подключение к интернету
- Первый запуск может занять больше времени, так как происходит загрузка модели
- Бот работает только в групповых чатах, не в личных сообщениях
